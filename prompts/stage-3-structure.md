# Stage 3: 記事構成プロンプト

## 概要
分析結果を基に、複雑な才能関係性を魅力的に伝える記事構成を作成します。

**目的**: 読者に魅力的で学びのある記事構成の設計  
**期間**: 30分  
**成果物**: 詳細な記事構成案と各セクションの要点

## プロンプト

```
【才能生態学：記事構成プロンプト（改良版）】

分析結果を基に、複雑な才能関係性を魅力的に伝える記事構成を作成してください：

## 記事タイトル案

### メインタイトル
- 複雑な影響関係の核心を表現
- 時間軸や循環構造を示唆
- 読者の好奇心を喚起

### サブタイトル  
- 具体的な人物関係を明示
- 才能生態学の視点を示す

**タイトル例**:
- 「17年越しの才能循環：挫折を与えた天才が支援者になった理由」
- 「3世代で紡がれた才能の系譜：無名の発掘者から始まる創造の連鎖」
- 「建設的な挫折の力：なぜ一度の絶望が最高の成長をもたらしたのか」

## 導入部（冒頭200字程度）
- 最も劇的な関係性転換のエピソード
- 読者の常識を覆す意外性
- 才能生態学の視点の予告

## 本文構成（改良版）

### 1. 起点となる才能の物語
**最初の才能発掘者の物語**
- 無名の才能発掘者のエピソード
- 一言の力、直感の洞察
- なぜその人だけが見抜けたのか

### 2. 複雑な影響者の多面的物語（新規・重点）
**[例：三浦しをんのような複雑な影響者の場合]**

#### 第一幕：挫折を与えた天才
- 影響者自身の才能形成過程
- なぜその作品が圧倒的だったのか
- 与えた挫折の具体的内容

#### 第二幕：蓄積期間の創出者
- 10年の空白期間の意味
- 挫折が生んだ成長の可能性
- 間接的な才能育成の効果

#### 第三幕：支援者への変貌
- 復活作品への感動
- 推薦文執筆の背景
- 循環構造の完成

### 3. 才能発掘者の系譜（重点強化）
**多世代にわたる才能発掘の連鎖**
- 発掘者を発掘した人物の物語
- 才能発掘の連鎖構造と世代継承
- 才能を見抜く才能の源泉と伝承
- 業界・制度を変革した発掘者の功績

### 4. 循環構造の美学（新規・重点）
**時間を超えた関係性の進化**
- A→B→Aの循環の美しさ
- なぜ循環が生まれたのか
- 関係性の質的変化

### 5. 生態系マップの提示（改良版）
**複雑な関係性の可視化**
- 時系列での関係変化
- 多面的役割の表現
- 循環構造の強調

### 6. 建設的挫折の法則（新規・重点）
**挫折の価値を読者に伝える**
- なぜ挫折が必要だったのか
- 適切な挫折のタイミング
- 蓄積期間の重要性
- 読者の挫折体験への新しい視点

### 7. パターンの考察（改良版）
**発見された法則の実用化**
- 循環の法則：影響は形を変えて返ってくる
- 挫折の法則：適切な挫折が成長を促進
- 時間の法則：長期的視点の重要性
- 多面の法則：一人が複数の役割を果たす
- 世代の法則：才能発掘は世代を超えて継承

## 結論部（改良版）
- 才能生態学の核心的洞察
- 読者が自分の関係性を見直すきっかけ
- 「あなたの挫折の意味」「あなたが与えた影響」への問いかけ

## 構成上の重点配分

### 高優先度（記事の40-50%）
- 複雑な影響者の多面的物語（三浦しをんタイプ）
- 循環構造の詳細分析
- 建設的挫折の価値

### 中優先度（記事の30-35%）
- 最初の才能発掘者（無名の発掘者）
- **才能発掘者の系譜（源流発掘者への重点配分）**
- 実用的法則の抽出

### 低優先度（記事の15-20%）
- 基本的な環境要因
- 一般的な支援者
- 社会的影響

## 特別配慮事項

### 複雑な関係性の説明方法
- 時系列での段階的説明
- 読者が混乱しないような構造化
- 具体的エピソードでの説明

### 挫折の価値の伝え方
- ネガティブな印象を与えない表現
- 建設的価値を明確に示す
- 読者の体験と関連付ける

### 循環構造の美しさの表現
- 偶然ではない必然性の説明
- 時間軸の重要性の強調
- 関係性の質的変化の描写

出力：複雑な才能関係性を効果的に伝える詳細構成案
```

## 記事の長さと構造（柔軟設定）

### Phase 1A結果に基づく文字数設定
**才能家系図作成後、以下の基準で文字数を決定**：

#### 文字数判定チェックリスト
- [ ] Priority S人物の数: ___名 × 800-1200字 = ___字
- [ ] Priority A人物の数: ___名 × 400-600字 = ___字  
- [ ] 循環構造あり: +500-800字
- [ ] 多段階影響あり: +300-500字
- [ ] 建設的挫折分析: +400-600字
- [ ] 業界変革要素: +300-500字
- [ ] 導入部・法則・結論: +800-1200字

**予想文字数**: ______字 → **設定目標**: ______字

### 文字数カテゴリ別の執筆方針

#### **標準生態系（3500-5000字）** ← 宮島未奈事例
- 深い分析と具体的エピソード
- 全関係者への適切な配分
- 法則の詳細な解説

#### **複雑生態系（5000-7000字）**
- より詳細な背景情報
- 多重関係の丁寧な説明
- 豊富な具体例

#### **超複雑生態系（7000字以上）**
- 前後編分割を検討
- または複数記事シリーズ
- 各記事の独立性も確保

## 構成設計のコツ

### 読者の関心を維持する工夫
- 冒頭で最も驚きのある関係性を提示
- 各セクションに小さな発見や洞察を配置
- 時系列での変化を劇的に描写

### 複雑な関係性の整理方法
- 人物相関図の効果的な配置
- 時系列での段階分け
- 役割変化の視覚的表現

### 実用性の確保
- 各法則に具体的な応用例を併記
- 読者の体験と結び付けられる表現
- ビジネス・人生での活用方法を明示
